IndexAwesome

by mochaorange

Installation

  1. Unzip IndexAwesome into a folder.
  2. Name that folder 
    resources
    .
  3. Upload the resources folder in to the root directory of your web server.
    Note: If you host multiple domains on your server, you'll need to upload one copy per domain. It needs to be in the root directory for the domain.

  4. Move the file 
    awesome.htaccess
    to the root directory as well, outside of the resources folder.
  5. Rename awesome.htaccess to simply 
    .htaccess
    .
  6. Enjoy!
